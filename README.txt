LEDTetris is a library you can use to develop Tetris on a LED Matrix of variable size and variable arrangement of the data lines of the LEDStrips.
It provides a class Matrix which does the calculating and manipulating the matrices.
This library also provides a board class, which generates a game board, this board class uses the Matrix class, so the programmer of the Tetris logic wont have to bother with the Matrix class.
